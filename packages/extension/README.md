# @conduit/extension

Chrome MV3 extension for the Conduit M0 spike. Connects to the Conduit Native
Messaging Host (`com.conduit.bridge`) and executes browser-side requests via
the Chrome DevTools Protocol.

## Build

From the repo root:

```bash
npm install --workspace @conduit/extension
npm run build --workspace @conduit/extension
```

The build emits a loadable, unpacked extension to
`packages/extension/dist/`.

## Load into Chrome

1. Build (above).
2. Open `chrome://extensions`.
3. Enable **Developer mode** (top-right toggle).
4. Click **Load unpacked** and select
   `packages/extension/dist/`.
5. Note the **extension ID** Chrome assigns. The options page also shows it.

## Wire up the NMH

The extension can only call `chrome.runtime.connectNative` if the NMH
manifest authorizes this extension's origin.

1. Locate your `com.conduit.bridge.json` NMH manifest (the path varies by OS;
   see `@conduit/nmh`'s `install:manifest` script).
2. Add this extension's origin to `allowed_origins`:

   ```json
   {
     "name": "com.conduit.bridge",
     "type": "stdio",
     "path": "/absolute/path/to/conduit-nmh",
     "allowed_origins": [
       "chrome-extension://<EXTENSION_ID>/"
     ]
   }
   ```
3. Reload the extension from `chrome://extensions`.
4. Click the toolbar action; the popup should report **NMH connected**.

## Sanity check without the NMH

If you haven't installed the NMH yet, the popup will show "NMH not
connected" but the **Test hello_world** button still works — it calls the
in-SW `hello_world` handler directly via `chrome.runtime.sendMessage`.

## Permissions

The MV3 manifest requests:

- `debugger` — required for CDP (`Accessibility.*`, `DOM.*`, `Input.*`).
  Triggers the yellow "DevTools is debugging this tab" banner; expected.
- `tabs` — to find a tab matching `urlPattern`.
- `nativeMessaging` — to connect to the NMH.
- `storage`, `alarms`, `scripting`, `cookies` — reserved for M1+ features.
- `host_permissions: ["<all_urls>"]` — broad for the spike; M1 narrows this.

## Layout

```
packages/extension/
├── esbuild.config.mjs       # bundles SW + popup + options, copies public/
├── public/
│   ├── manifest.json
│   ├── popup.html
│   ├── options.html
│   └── icons/               # generated by scripts/gen-icons.mjs at build time
├── scripts/gen-icons.mjs    # zero-dep PNG generator (16/48/128)
└── src/
    ├── sw/
    │   ├── index.ts             # service worker entry
    │   ├── nmh-port.ts          # native messaging connect + reconnect
    │   ├── debugger-util.ts     # CDP attach/sendCommand wrappers
    │   ├── ax-tree.ts           # CDP node list -> AxNode tree
    │   ├── url-match.ts         # glob -> regex
    │   ├── errors.ts
    │   └── handlers/
    │       ├── hello-world.ts
    │       ├── extract-ax-tree.ts
    │       └── click-by-role-name.ts
    ├── popup/index.tsx
    └── options/index.tsx
```
